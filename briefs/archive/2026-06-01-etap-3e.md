Data: 2026-06-01
Status: ✅ Potwierdzone przez użytkownika

## Zakres

Plik: `WysiwygHtml.vue`

1. `StarterKit.configure({ heading: { levels: [1, 2, 3, 4, 5, 6] } })` — odblokowanie H4-H6
2. Computed `currentHeadingLevel` — sprawdza aktywny heading i zwraca '0'-'6'
3. Funkcja `setHeading(level)` — `setParagraph()` dla 0, `setHeading({ level: n })` dla 1-6
4. `<select class="toolbar-select">` z opcjami Normalny/H1-H6 zastąpił osobne przyciski H1/H2/H3
5. CSS `.toolbar-select` w `<style scoped>`
