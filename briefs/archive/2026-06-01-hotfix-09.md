# HOTFIX 9: brakujące klasy `md:grid-cols-2` i `md:grid-cols-3` ✅ ZAMKNIĘTY

Data: 2026-06-01
Status: ✅ Wykonane

## Problem

Tailwind JIT nie wygenerował klas `md:grid-cols-2` i `md:grid-cols-3` przy pierwszym buildzie — race condition (build uruchomiony przed zapisem pliku widoku).

## Fix

Tylko jedna komenda: `npm run build`

## Kluczowy wniosek

Tailwind JIT (v4) musi widzieć pełne nazwy klas w kodzie źródłowym — klasy w ciągach interpolowanych nie są wykrywane. Przy race condition budowania wystarczy ponowny `npm run build`.
